# AI Code Reviewer
An AI-powered code review assistant using OpenAI GPT.

## Features
- Supports Python, JavaScript, Java, C++
- Detects bugs, performance issues, security flaws
- Suggests improvements

## Setup
1. Clone repo
2. `cd backend` and install dependencies: `pip install -r requirements.txt`
3. Add your OpenAI API key in `.env`
4. Run `python app.py`
5. Open `frontend/index.html` in browser
